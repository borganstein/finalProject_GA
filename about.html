This is the final project of a General Assembly course. Using HTML, CSS and Javascript to create a flashcard site to Learn German.

I am bilingual in German (and obviously English) but living in Australia for the last 25 years means that I struggle with my German. This site, I hope will help me (and possible others) to learn some German. Also, just a fun way to play around with some basic HTML, CSS and JS.

The colour scheme for this website is influenced by the Canton of Zurich, Switzerland whose primary colours are white and blue. I grew up in Zurich and so it has a special place in my heart

Hope you enjoy and please hit me up if you have any questions or helpful tips.
